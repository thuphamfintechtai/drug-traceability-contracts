# drug-traceability-contracts



## Cách compile Contract

```shell
cd Contracts 
npx hardhat compile
```

## Cách test contract 

```shell
cd Contracts 
npx hardhat test
```


## Cách Deploy Contract lên mạng của Pione zero Chain 


```shell
cd Contracts 
npx hardhat ignition deploy ./ignition/modules/deploy.ts --network pione
```
